Week 7 Notes

  • Functions can be passed around in the same way as every other value.
  • Functions have a length property that returns the number of parameters the function has.
  • Here is an example of that using the "square()" function:

function square(x) {

return x*x;

}

square.length

Outcome:

<< 1

  • With functions we can make custom properties like one for descriptors or anything else.
  • Here is an example of the function square getting a customer property named description:

square.description = 'Squares a number that is provided as an argument'

Output:

'Squares a number that is provided as an argument'

  • Ajax sounds like a weird term but it actually is a technique that allows web pages to communicate together with a server.
  • We can fetch a API like this:

fetch('https://example.com/data')

  • This is very useful for responsive interfaces and receiving information.
  • Going further into Ajax we can fetch information, send information, and form data from that.
