Week 9 Notes

  • The window object in JavaScript can be used to change the environment of the web page.
  • The BOM is a group of properties and methods that can help us in situations with pop up windows and dimensions of a screen.
  • Here is an example of a BOM environment:

// from within the global scope

const global = this;

  • The document object can help with a page loaded inside the window.
  • We can even display text as a string right on the page with the write variable like this:

document.write('Hello, world!');

Output: Hello, world!

  • In HTML we have things called API's. They can help us pull data from websites and keep it organized.
  • We can use a property called localStorage. We can store data and methods here like this:

localStorage.setItem('name', 'Walter White');

  • We can also draw with an element called canvas
  • Here is an example of using the canvas element to display some shapes:

context.fillStyle = '#0c0'; // a blue fill color

context.font = 'bold 26px sans-serif';

context.fillText('Hello', 20, 200);
