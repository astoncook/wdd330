Week 8 Notes

  • Transformation properties help us do a number of things like to translate, rotate, scale, and/or skew any element on the page.
  • Translation allows us to move elements left right up and down. This can help with more precise movement.
  • Here is an example of the translate function with a x and a y:

transform: translate(45px, -45px);

  • This is pretty simple and can help us in numerous circumstances.
  • Another few functions that can help us are called Scaling, Rotating, and Skew
  • We have one called Transitions that can help with animations.
  • Here is an example of an animation displaying a bike:

.ad-ad3 :after {

content: '';

width: 90px;

height: 92px;

background-image: url(../images/bike_sprite.png);

display: block;

margin: auto;

}

@keyframes bike {

0% {

background-position: 0 0;

}

100% {

background-position: -360px 0;

}

}

  • In HTML5 they have more elements that can help with videos. Canvas was created by Apple and can help a user draw through JavaScript.
  • Here is a simple example of a box:

.myCanvas {

border: dotted 2px black;

}

  • This is very useful and can be a fun skill to learn for future projects.
  • In the world of customization, there is much more to talk about like SVG, and Drag and Drop. These are also very useful for good-looking projects and things of the sort.
