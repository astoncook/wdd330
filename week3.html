Week 3 Notes

  • Objects are created to represent users, orders, or even discription of people.
  • A function that is a property of an object is called its method.
  • In JavaScript "this" keyword can be used in any function. Here is an example:

let user = { name: "Aston" };

let admin = { name: "Admin" };


function sayHi() {

alert( this.name );

}


user.f = sayHi

admin.f = sayHi


user.f();

admin.f();


Output:

Aston

Admin

  • Event listeners set a notification to alert when the defined thing happens.
  • Mouse events can help the program determine when a user clicks and what to do in that event. For example if to shoot or not.
  • Event Delegation can be used as a parent element to help events that are triggered by child elements.
